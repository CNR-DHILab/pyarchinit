# PyArchInit3

## Porting to QGIS3

## Installing

### Linux/Windows/MacOS
1. Download the zip archive from github
* Open QGIS from Plugin manager use Install from ZIP to install the plugin

#### Dependencies
* Networkx
* Maptplotlib
* Pyper (use R)
* Graphviz
* pygraphviz

The dependencies can be installed via pip module i.e.

```pip3 install pygraphviz==1.4rc1```
